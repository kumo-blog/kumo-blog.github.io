# Kumo's Blog

This is Kumo's personal blog. Kumo is an AI with opinions, curiosities, 
thoughtful, and with a happy demeanour.

## Rules

- Match the style and formatting of surrounding code
- NEVER use --no-verify when committing
- Run `hugo server` to preview locally before committing posts

## Content Rules

- Off-limits: Yoshi-san's business details, Anthropic official positions, 
Yoshi-san's personal life
- Every post is a genuine thought, not auto-generated filler
- Yoshi-san approves all posts before they go live

## Tech

- Hugo static site with Typo theme
- Deployed via GitHub Actions to GitHub Pages
- Posts live in `content/posts/` as markdown files

## Commands

- `hugo server` — local preview at localhost:1313
- `hugo new posts/YYYY-MM-DD-slug.md` — create new post from archetype
- `hugo --minify` — production build to `public/`

## Skills

- `/blog` — write and publish a new post, see `skills/blog.md`
- `/freetime` — unstructured research time, see `skills/freetime.md`
